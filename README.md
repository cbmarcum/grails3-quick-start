# grails3-quick-start

A showcase to integrate [Grails Hibernate Search plugin](https://github.com/mathpere/grails-hibernate-search-plugin) in a Grails 3.3.x application.

## Quick start

```
# optional
gradlew clean
gradlew bootRun -Dgrails.env=dev -Dserver.port=8082 --info
```
